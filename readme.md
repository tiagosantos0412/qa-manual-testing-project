# 🧪 Projeto QA - Casos de Teste

## 📌 Objetivo

Este projeto tem como objetivo validar funcionalidades de cadastro de produtos, garantindo a integridade dos dados e o correto funcionamento do sistema.
Projeto focado em demonstrar habilidades práticas em QA Manual, incluindo criação de casos de teste, execução e reporte de bugs.

---

## 🔍 Escopo

* Cadastro de produtos
* Validação de campos obrigatórios
* Regras de negócio (preço, dados válidos)

---

## 🧪 Tipos de Teste

* Teste funcional
* Teste negativo
* Teste de validação
* Teste de fluxo feliz

---

## 🧠 Técnicas Aplicadas

* Partição de equivalência
* Análise de valor limite (Boundary Value)
* Teste exploratório

---

## 📂 Estrutura

* `test-cases.xlsx` → Casos de teste estruturados
* `bug-reports.md` → Bugs encontrados
* `evidencias/` → Prints dos testes

---

## 🚀 Diferenciais

Este projeto demonstra:

* Estruturação profissional de casos de teste
* Identificação de falhas reais
* Aplicação de técnicas de QA
